# Stamped — Website

Marketing site for [Stamped](https://stamped.sg) — AI job documentation for Singapore managing agents.

## Deployment

This site is deployed via GitHub Pages from the `main` branch.

**Live URL:** https://thestealthyworker.github.io/stamped-website

**Custom domain (when ready):** https://stamped.sg — add a `CNAME` file containing `stamped.sg` and point your DNS A records to GitHub Pages IPs.
